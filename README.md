# Biblioteca-online

Proyecto final de **Programación Web** desarrollado con **HTML, CSS, Bootstrap, JavaScript, PHP y MySQL**.  
Este sistema permite consultar el listado de libros y autores de una librería online, además de enviar mensajes mediante un formulario de contacto.

## Descripción

La aplicación fue creada con el objetivo de poner en práctica los conocimientos adquiridos durante el curso, integrando desarrollo web front-end y back-end con conexión a base de datos.

El sistema está completamente en español y cuenta con una interfaz sencilla, moderna y fácil de usar.

## Funcionalidades

- Página principal con menú de navegación
- Listado de libros disponibles
- Listado de autores registrados
- Formulario de contacto
- Registro de mensajes en la base de datos
- Validación de formulario con JavaScript
- Diseño responsivo con Bootstrap
- Conexión a MySQL usando PDO

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript
- PHP
- MySQL
- PDO
